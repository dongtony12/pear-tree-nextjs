# 배나무 NEXT.js
배달 파티를 구성해 부담이 되는 배달료를 줄여보세요.

## Environment
node 16+ / yarn 3+

## Run project

setup

```
cp .env.example .env.local

yarn
```

start storybook

```
yarn storybook
```

start project

```
yarn dev
```

## Commit rules
* ADD | install some dependency
* CREATE | implement a new feature
* UPDATE | update some feature / logic / dependency
* DELETE | delete library or feature or log
